# Overview

Stagehand is an AI web automation framework that uses natural language instructions to control browsers and extract data.

### Overview

The Stagehand integration connects AI browser automation with Steel's browser infrastructure. This enables:

- Control Steel browser sessions with natural language commands

- Extract structured data using AI

- Automate complex workflows with minimal code

- Handle dynamic content and modern web apps

Combining Stagehand with Steel gives you reliable automation with anti-bot capabilities, proxy support, and sandboxed environments.

### Requirements & Limitations

- **OpenAI API Key**: Access to the OpenAI API

- **Steel API Key**: Active subscription to Steel

- **Node.js or Python Environment**: Support for Stagehand in your preferred language

- **Supported Environments**: Works best with Steel's browser environment
