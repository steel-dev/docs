Hey everyone — this week brings new deep research examples to the cookbook, an ecosystem addition, and a round of corrections to the Files API docs.

### ⭐ New

*   Added deep research examples to the cookbook with updated agent workflows.

### 🔧 Improvements

*   Added Hermesforge Screenshot API to awesome-web-agents.

### 🐛 Bug Fixes

*   Corrected the Files API docs: upload endpoints are `POST /v1/sessions/{id}/files` and `POST /v1/files` (no `/upload` suffix), the session archive endpoint is `GET /v1/sessions/{id}/files.zip`, list responses now use a `{ data: [...] }` envelope, the SDK exposes `client.sessions.files.downloadArchive(...)`, and global file uploads return `path` rather than `id`.
